## Agent URLs

Every agent template has a corresponding unique URL. 

You can find agent templates on the [WithAI website](https://www.getwith.ai) or via the [List Templates API](/api-reference/list-templates). 

## Configuring Agent URLs with Context

Each agent URL can be modified with a URL-encoded `context` query argument specifying more information about the task the agent should perform. The `context` value can be any natural language string. 

For example, this agent URL pre-fills the agent template with a specific task:

```json
https://www.getwith.ai/automate-instagram?context=Respond%20to%20customer%20messages%20thanks%20for%20reaching%20out%20and%20tell%20them%20I%20will%20get%20back%20to%20them%20soon
```


You can generate pre-configured URLs directly from the WithAI website and get specifications about the parameters that can be configured via the [Template Details API](/api-reference/template-details).
